使用Python语言编写程序时，我们常常需要导入Python模块或包（类似于C语言中的头文件），而Python的导入模块方法非常多，本文就是写的有关Python导包的内容。

Python的导入主要分为一下五类：

  • 常规导入 (regular imports)
  • 使用from语句导入 (import from )
  • 相对导入 (relative imports)
  • 可选导入 (optional imports)
  • 本地导入 (local imports)
